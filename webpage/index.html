About

Money Manager is an open-source personal finance application. Its goal is to support every type of account a person can have — banking, pensions, investments, crypto — while automating data entry as much as possible, which is intended to be its distinguishing feature.

It is also an exploration of what can be achieved with Kotlin Multiplatform and Compose Multiplatform: a single codebase that ships a desktop (JVM) app and an Android app.

Features

✅ Working today

  • Accounts, categories and transactions
  • Basic transfers between accounts
  • Basic CSV importing
  • Audit history
  • Cloud-backed databases (Google Drive, encrypted)
  • Desktop (JVM) app
  • Android app
  • Automated deploys

🚧 Planned

  • Charting and data exploration
  • Forecasting and budgeting
  • Multi-user support
  • Reconciliation support
  • iOS and Web (JS/WASM) versions
  • AI classification of transactions
  • Export to GnuCash

Core concepts

EntityDescription
AccountFinancial accounts — checking, savings, credit card, cash, investment
CategoryTransaction categories with a parent/child hierarchy
TransactionIncome, expense, and transfer records

Monetary amounts are handled with exact arithmetic only — never floating point. Values are stored as integers scaled per currency, and all parsing and calculation goes through BigDecimal, so amounts are always precise.

Getting started

Clone the repository and build (requires JDK 25):

git clone https://github.com/NikolayMetchev/money-manager.git
cd money-manager
./gradlew build --console=plain

Run the desktop app:

./gradlew :app:main:jvm:run --console=plain

Install the Android debug build on a connected device or emulator:

./gradlew :app:main:android:installDebug --console=plain

Project status

Money Manager is in beta. The database schema is still evolving and migrations are not yet implemented — local databases are recreated on schema changes. Once the tool reaches a usefully stable state, contributions will be very welcome.
